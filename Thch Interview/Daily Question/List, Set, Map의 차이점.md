# List, Set, Map의 차이

## List

- 순서가 있다.
- 중복이 허용된다.
- 인덱스를 통한 데이터에 접근이 가능하다.


## Set

- 순서가 없다.
- 중복이 허용되지 않는다.
- null이 입력가능하나 null또한 중복이 허용되지 않는다.
- 인덱스가 따로 존재하지 않기때문에 Iterator를 이용해 접근해야한다.


## Map

- Key & Value 구조이다.
- Key:
  - 중복이 허용되지 않는다.
  - 인덱스가 존재하지 않고 Key를 이용해 접근한다.
  - Iterator를 이용해 접근할 수 있다.
- Value:
  - Key값에 따라 다니는 값이다.
  - 중복이 가능하다.
